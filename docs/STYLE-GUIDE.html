01 — IDENTITY

A blue → red brand. With yellow trim.

The Gala visual identity is built on a two-color story: deep brand blue (#2858d6) and crimson red (#CB262C), set against a near-black DEF navy (#0b1b3c) ground. The gradient runs directly from blue to red without a purple bridge — keeping the brand readable, optimistic, and unmistakably civic. DEF yellow (#ffc24d) is the foundation's signature accent and earns a small but real role on Gala surfaces: dividers, scrollbars, hairline icons, dots, and italic flair words on dark grounds. Never on paper. Never on text the user has to read.

03 — GRADIENTS

The signature moves.

Five gradient patterns carry the brand. All shimmering gradients use the same 4-second linear animation (goldShimmer) with background-size: 200% auto — the timing and easing must match across surfaces or the brand reads as out of sync.

Brand Gradient — Headline Text
The 125° five-stop wash. Used as a clipped background on italic display text (the Nelson. / 2026. moment) and as the 3px top strip on cards. Animated.
linear-gradient(125deg, --gala-blue, --gala-red, --gala-blue-light, --gala-red, --gala-blue)
Horizon Strip — Brand Bar
90° four-stop. The 3px bar that runs across the very top of every Gala surface and beneath the masthead. Email mastheads. Animated.
linear-gradient(90deg, --gala-blue, --gala-red, --gala-blue-light, --gala-red)
Stat Number — Red-First Variant
Reversed stop order. Used on the big shimmering stat numbers in the navy stats banner. Always paired with a subtle blue drop-shadow glow.
linear-gradient(90deg, --gala-red, --gala-blue-light, --gala-red, --gala-blue)
Red CTA — Primary Button
Static 135° red-on-red-deep. Pulses scale, never color. Sponsor portal "Place" button uses this exact fill.
linear-gradient(135deg, --gala-red, --gala-red-deep)
Tier Pill — Sponsor Levels & Pills
Static 135° blue-to-red. Sponsor tier price pills, active nav pills, "Invite" callouts on the portal. The two-color story at its simplest.
linear-gradient(135deg, --gala-blue, --gala-red)

05 — MOTION

Hum, don't flourish.

Two core animations carry the brand. No entrance animations (curtain reveals, letter rises, halo drifts, ken-burns zooms) on hero sections — these read as Squarespace and break the editorial seriousness. Static heroes, animated trim.

Animation · goldShimmer

The brand pulse. 4-second linear infinite loop sliding the gradient background-position from 0% to 200%. Applied to every animated gradient surface (strips, headline text, stat numbers, top trim). All four must share this exact timing or the brand desyncs visually. 

animation: shimmer 4s linear infinite;
background-size: 200% auto;

@keyframes shimmer {
  0%   { background-position: 0% center; }
  100% { background-position: 200% center; }
}
Animation · ctaPulse

Scale-only pulse on the primary red CTA. 2.6-second ease-in-out, pauses on hover. The button breathes (scale 1 → 1.03) without the box-shadow expanding — color never animates. Used on btn--red only; never on outlines, ghosts, or text links. 

animation: ctaPulse 2.6s ease-in-out infinite;

@keyframes ctaPulse {
  0%, 100% { transform: scale(1); }
  50%      { transform: scale(1.03); }
}
Hover & focus easing

All component-level transitions use a custom ease for a slight overshoot feel. Card lifts on hover use transform: translateY(-3px) + box-shadow. Buttons use shadow + filter only — no translate. 

--ease: cubic-bezier(0.22, 0.61, 0.36, 1);
transition: transform 260ms var(--ease),
            box-shadow 260ms var(--ease),
            border-color 260ms var(--ease);

07 — DEF YELLOW

The third voice.

Yellow is the smallest brand color, but it is no longer just "trim." It earns a real if minor role across the system — provided it stays on dark grounds and never tries to carry text the user has to read. The rule of thumb: if removing the yellow would leave the surface still legible and complete, yellow is being used correctly. If removing it breaks understanding, you've asked too much of it.

Eyebrow on navy
— THE LINEUP

Yellow eyebrows on navy. Replaces the all-red eyebrow on dark grounds for variety. Paper grounds keep red eyebrows.

Italic flair · on navy only
Lights, camera, action.

Italic display word in yellow as an alternative to the blue→red gradient. Use sparingly — once per page. Forbidden on paper (illegible).

Hairline divider

Yellow-fade hairline between sections on navy. Replaces the white rule for visual rhythm.

Gold-tier sponsor dot
Gold tier sponsor

The only place yellow gets to literally represent the gold-tier brand level. Glowing dot, never a filled tier card.

Hairline icon stroke

Yellow 1.6px stroke icons on navy grounds. Section markers, decorative glyphs. Never for primary action icons (those stay white or red).

Scrollbar · scroll this page
Custom yellow scrollbar — already active on this page.

Yellow scrollbars on Gala surfaces. A tiny, almost-subliminal moment of brand presence. The very definition of trim.

The yellow test

Before adding yellow to a new surface, ask: does removing it break the design or just dim it slightly? If removing it breaks anything, find another color. Yellow is jewelry, never structure.

09 — RULES

Do & don't.

Hard-won. Most of these come from specific surfaces that shipped wrong.

DO
  • Pair Fraunces italic with the brand gradient for headline emphasis (the Nelson. moment).
  • Use --gala-red for primary CTAs, --gala-blue for selection and progress states.
  • Anchor every Gala surface with the 3px shimmering strip at the very top.
  • Keep the DEF navy ground (#0b1b3c) as your darkest value. Don't go pure black.
  • Apply the radial blue + red atmosphere wash on any dark surface (heroes, stats, footers).
  • Use --def-gold-400 for eyebrows, italic flair, hairline dividers, dots, and scrollbars on navy grounds only.
  • Static heroes. The image doesn't move. The trim moves.
  • Maintain 4-second shimmer timing across every animated gradient surface.
DON'T
  • Use light purple (#a8b1ff) for status numbers, "yours" markers, or any UI surface.
  • Bridge the gradient through magenta. The brand goes blue → red, not blue → purple → red.
  • Apply yellow to body text, buttons, numbers, status, or any text on paper grounds. Gold-on-paper is illegible — the rule was earned the hard way.
  • Animate hero entrances (curtain reveals, letter rises, ken-burns). Static heroes only.
  • Ship Source Serif on Gala surfaces. Fraunces is the canonical display font.
  • Use Inter for display headlines. Inter is body only.
  • Color-animate buttons. Scale or filter only.
  • Use sed on production HTML for CSS changes. CSS edits are appended and reviewed.
10 — DROP-IN

The tokens, ready to ship.

Paste this block at the top of any Gala stylesheet, or set as the :root in a new component. Both src/brand/styles.css and src/brand/tokens.js on the gala repo should converge on these values. 

/* Gala 2026 — canonical tokens. Mirror from daviskids.org. */
:root {
  /* Brand reds */
  --gala-red:        #CB262C;
  --gala-red-light:  #e84e53;
  --gala-red-deep:   #a01f24;

  /* Brand blues — these REPLACE all purple/indigo tokens */
  --gala-blue:       #2858d6;
  --gala-blue-light: #4a7df0;  /* the new "yours" color */
  --gala-blue-deep:  #1a3aa3;

  /* DEF navy — aligned to canonical daviskids.org tokens */
  --navy-950: #0b1b3c;  /* WAS #0d1b3d — corrected */
  --navy-900: #122a57;
  --navy-800: #1c3a78;
  --navy-700: #24508f;

  /* DEF gold — trim and accents on navy ONLY, never on paper */
  --def-gold-300: #ffd77f;  /* soft / glow / dividers */
  --def-gold-400: #ffc24d;  /* canonical accent */
  --def-gold-500: #f5a623;  /* deepest */
  --def-sky-300:  #8bcfff;  /* complement, sparing use */

  /* Type */
  --font-display: "Fraunces", Georgia, serif;
  --font-sans:    "Inter", system-ui, sans-serif;

  /* Signature gradients */
  --grad-brand: linear-gradient(125deg,
    var(--gala-blue), var(--gala-red),
    var(--gala-blue-light), var(--gala-red), var(--gala-blue));

  --grad-strip: linear-gradient(90deg,
    var(--gala-blue), var(--gala-red),
    var(--gala-blue-light), var(--gala-red));

  /* Yellow trim — for hairlines and scrollbars */
  --grad-gold-soft: linear-gradient(180deg,
    var(--def-gold-300), var(--def-gold-500));

  /* Motion */
  --ease: cubic-bezier(0.22, 0.61, 0.36, 1);
}

@keyframes shimmer {

  100% { background-position: 200% center; }
}
